================================================================================
  JOURNEY THROUGH SCRIPTURE - FREE ASSET DOWNLOAD QUICK START
================================================================================

Your game is ready for assets! All the infrastructure is in place.
Now you just need to download free open-source assets.

TIME NEEDED: 30-45 minutes to download everything
COST: $0 (all free and legal!)

================================================================================
  STEP 1: DOWNLOAD CHARACTER SPRITES
================================================================================

1. Go to: https://opengameart.org/art-search-advanced?field_art_tags_tid=14
2. Look for "LPC Universal Spritesheet" or "LPC Medieval Fantasy"
3. Click the asset
4. Download the PNG file (it's free!)
5. Extract if needed
6. Copy to: mygame/web/static/webclient/assets/sprites/characters/
7. Rename to: player.png

REPEAT for NPCs:
  - Download LPC Medieval Fantasy Characters pack
  - Copy as: npc_priest.png, npc_merchant.png, npc_elder.png, pilgrim.png

================================================================================
  STEP 2: DOWNLOAD TILESET (Palace Floors)
================================================================================

1. Go to: https://opengameart.org/art-search-advanced?field_art_tags_tid=40
2. Filter by "Tile Set" art type
3. Look for "LPC Dungeon" or "32x32 Indoor Tiles"
4. Download the PNG tileset
5. Copy to: mygame/web/static/webclient/assets/tiles/palace/floor1/
6. Rename to: indoor_tileset.png

For additional floors, download more tilesets:
  - Copy to floor2/ and floor3/ folders
  - Or reuse the same tileset in each folder

================================================================================
  STEP 3: DOWNLOAD ENEMY SPRITES
================================================================================

1. Go to: https://opengameart.org/art-search-advanced?field_art_tags_tid=13
2. Look for "LPC Monsters" spritesheet
3. Download PNG
4. Copy to: mygame/web/static/webclient/assets/sprites/enemies/
5. Name as: deceiver.png, demon.png, undead.png

================================================================================
  STEP 4: DOWNLOAD ITEM SPRITES (Optional but Recommended)
================================================================================

1. Go to: https://opengameart.org/art-search-advanced?field_art_tags_tid=28
2. Look for "LPC Items" or "LPC Items and Objects"
3. Download PNG
4. Copy to: mygame/web/static/webclient/assets/sprites/items/
5. Name as: weapons.png, potions.png, quest_items.png, treasure.png

================================================================================
  STEP 5: DOWNLOAD BACKGROUND MUSIC (Optional)
================================================================================

Music makes a HUGE difference!

1. Go to: https://opengameart.org/art-search-advanced?field_art_tags_tid=27
2. Search for "Medieval Dungeon" or "Fantasy Music"
3. Download OGG or MP3
4. Copy to: mygame/web/static/webclient/assets/audio/music/
5. Name as: medieval_exploration.ogg, boss_battle.ogg, ambient_palace.ogg

Alternative: https://incompetech.com/ (excellent royalty-free music)

================================================================================
  STEP 6: DOWNLOAD SOUND EFFECTS (Optional)
================================================================================

1. Go to: https://freesound.org/
2. Search for: "sword strike", "magic spell", "door open", "chest open"
3. Filter by: Creative Commons license
4. Download WAV files
5. Copy to: mygame/web/static/webclient/assets/audio/sfx/
6. Name as: combat_hit.wav, magic_spell.wav, door_open.wav, chest_open.wav
