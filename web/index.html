About Journey Through Scripture

Journey Through Scripture is a graphical MUD (Multi-User Dungeon) set in a biblical fantasy world.

Features

  • Explore lands inspired by biblical scripture
  • Battle mythical creatures from ancient lore
  • Receive guidance from divine messengers
  • Progress through a rich character development system
  • Experience immersive graphics and sound

Technology

Built with Evennia MUD framework, HTML5 Canvas, and Web Audio API

Credits

Created as a learning project combining classic MUD gameplay with modern web technologies
